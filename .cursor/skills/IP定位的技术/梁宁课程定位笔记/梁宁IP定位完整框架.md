## 核心公式

### 用户价值
```
用户价值 = 新体验 - 旧体验 - 替换成本
```

### 产品价值
```
产品价值 = 功能价值 + 情绪价值 + 资产价值
```

### 爆品公式
```
爆品 = 技术/供应链创新 × 爆发品类 × 新流量
```

### 增长飞轮
```
核心价值 → 更多用户 → 更多资源 → 更好价值
```
