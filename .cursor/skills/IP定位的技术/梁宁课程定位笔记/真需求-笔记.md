## PART ONE：价值与需求

### 第一章：功能价值与效率需求

**功能价值的四个模型**：

| 模型 | 说明 | 例子 |
|------|------|------|
| 原材料/劳动力 | 卖原材料+人工 | 农产品、袜子 |
| 专利/IP | 卖技术/知识产权 | 西药、芯片 |
| 供应链/平台 | 卖交付系统 | 京东、美团 |
| 基础设施 | 赋能能力 | 电力、互联网 |

**供应链三代演进**：
- 初代（有啥卖啥）- 丝绸之路、京东
- 二代（没啥造啥）- 果链、特链
- 三代（链到底）- SHEIN（小单快反）

---

### 第二章：情绪价值与情感需求

**情绪产生机制**：
```
生理唤起 + 认知标记 = 情绪
```

**情绪价值付费点**：
1. **心理账户** - 用户愿意在哪类账户花钱
2. **情绪价值三要素**：
   - 生理唤起
   - 认知标记
   - 心理账户

---

### 第三章：资产价值与投资需求

**资产价值特征**：
- 越涨越有人买（投资品特性）
- 奢侈品策略：只涨价不降价

**价值公式**：
```
产品价值 = 功能价值 + 情绪价值 + 资产价值

奢侈品 = 原材料稀缺 + 彰显性 + 资产价值
```

---

### 第四章：产品创新-价值组合

**创新逻辑**：
```
创新 = 新组合 × 品类机会
```

**品类进化**：
- 新物种 - 全新的需求和解决方案
- 品类分化 - 从旧品类中分化
- 品类融合 - 跨品类整合

---

### 第五章：品牌价值

**品牌层次**：

| 类型 | 公式 | 特征 |
|------|------|------|
| 白牌 | 产品价值+商标 | 无须被记住 |
| 老牌 | 产品价值+熟悉感 | 10年+ |
| 网红 | 产品价值+新鲜感+话题度 | 声量 |
| 大牌 | 产品价值+辨识度+情感唤起 | 记忆 |
| 头牌 | 产品价值+辨识度+行业领导力 | 大宗师 |

## PART TWO：从分歧到共识

### 第六章：感知的分歧（KANO模型）

**KANO模型四象限**：

| 属性 | 定义 | 例子 |
|------|------|------|
| 期望属性 | 有则满意，无则不满 | 味道、分量 |
| 魅力属性 | 超出预期，有则惊喜 | 漂亮老板娘 |
| 必备属性 | 默认应该有 | 卫生 |
| 无差异属性 | 有没有都无所谓 | 瓶子设计 |
| 反向属性 | 有反而不满 | 香菜 |

---

### 第七章：想象的分歧（用户人设）

**用户画像 vs 用户人设**：
- 用户画像：一个人没有什么，就需要什么（刚需时代）
- 用户人设：一个人的长项，就是他的需求（丰饶时代）

**人设公式**：
```
人设 = 固定人物分类 + 记忆锤 + 外在形象特点 + 性格 + 特长
```

**改变公式（贝克哈德哈里斯）**：
```
不满 × 愿景 × 第一步 > 改变的阻力
```

---

### 第八章：场景的分歧（产品场景匹配）

- **场景更具体，产品更有力**
- 案例：亲子游泳馆 → "爸爸带孩子学游泳"场景
- 黄金场景 = 值得200%打磨的场景

---

### 第九章：利益的分歧

**利益相关人地图**：
```
     影响力
      ↑
高   │关键人物    │合作伙伴
     │            │
     │            │
     │            │
低   ├────────────┼──────────→ 利益
     │            │
     │            │
     │反对者      │受损者
```

**五种共识**：
1. 内部共识：利益即态度
2. 客户共识：阵营管理
3. 市场共识：驱动利益链
4. 社会共识：危机公关、ESG

---

### 第十章：关系（共识的成果）

**关系的四个次第**：
| 次第 | 特征 |
|------|------|
| 需要 | 有用 |
| 喜欢 | 愉悦 |
| 认同 | 价值观一致 |
| 归属 | 共同体 |

## PART THREE：模式

### 第十一章：能力系统

**四个能力系统**：
1. 交易能力 - 卖货
2. 产品能力 - 做产品
3. 组织能力 - 带团队
4. 资本能力 - 融资

**模式第一问**：拿谁的钱？
- toVC模式：从资本拿钱
- to市场模式：从用户拿钱

**案例**：
- 瑞幸：从资本驱动 → 产品驱动
- 每日优鲜：失败（未从用户处获利）

---

### 第十二章：变现逻辑

**套利空间**：
- 信息套利
- 资源套利
- 能力套利

---

### 第十三章：分配机制

**原则**：
- 资源向变现更快的地方聚集
- 对顶尖人才给予奖赏

---

## PART FOUR：求真创造者的精神

### 第十四章：应然与实然

**应然 vs 实然**：
- 应然：大脑里事情应该是怎样的
- 实然：事情实际上是怎样的

**两种路径**：
- 草根（李兴平）：先实然，后应然
- 精英（王小川）：先应然，后实然

**弱者道之用**：
- 强者看到的是"我我我"
- 弱者看到的是真需求
- 真需求是互联网的天道

---

### 第十五章：第一性原理

**核心**：
- 回到最基本的常识
- 不被既有框架限制
- 用物理原理解决问题
