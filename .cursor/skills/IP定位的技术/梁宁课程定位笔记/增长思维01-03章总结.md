## 第一章：在生死进退中增长

### 核心观点

1. **增长的三种状态**
   - 维持：年增长 ≈ GDP增长率（跑输大盘）
   - 增长：年增长 ≈ GDP的2倍（跑赢大盘）
   - 爆发式增长：年增长 ≈ GDP的4倍以上（需要红利）

2. **爆发式增长来自空白**
   - 空白 = 曾经束缚市场的限制条件改变
   - 打破界限，继续增长
   - 每当天花板打开，新空间出现

3. **企业的生死**
   - 死 = 主导者失去战斗意志
   - 退 = 善败者不亡
   - 核心团队是剩下来的，不是招募来的

4. **关键认知**
   - 视野和自信决定格局
   - 亚马逊不断打开天花板，进入无人区
   - 善败者不亡：保护核心资产，主动撤退

### 与IP定位的关联

| 维度 | IP定位中的应用 |
|------|----------------|
| 增长思维 | IP的增长来自空白，找到被忽视的用户群体 |
| 战斗意志 | 做IP需要持续的战斗意志，不放弃 |
| 善败 | 及时调整方向，保护核心能力 |

## 第二章：从草莽到腰部玩家 - 运营生态位

### 核心观点：企业四种位置

| 位置 | 核心动作 | 特征 |
|------|----------|------|
| **草莽** | 冲 | 看到机会先干，速生速死 |
| **腰部** | 扎根 | 找到生态位，构建价值增量 |
| **头部** | 收 | 抢制高点，终结战争 |
| **顶级** | ？ | 跨越周期 |

### 生态位选择

- 生态位 = 生态系统中的角色和地位
- 附着于什么生态，决定生死
- 案例：云海肴 vs 茶马古道
  - 云海肴：选择shopping mall（城市化进程）
  - 茶马古道：选择后海（受季节影响）
- 结果：云海肴300家分店，茶马古道消失

### 腰部企业的关键

1. **看到生态系统变化**
2. **做出选择**
3. **主动投入，构建价值增量**

### 案例：格瓦拉电影票

- 找到生态位：在线选座+自助取票机
- 构建价值增量：完成体验闭环
- 被巨头覆盖：BAT入场后无法抵抗
- 全身而退：几十亿出售

### 与IP定位的关联

| 维度 | IP定位中的应用 |
|------|----------------|
| 生态位 | 找到自己的细分赛道和用户群体 |
| 腰部思维 | 不追求头部，先找到一个可持续的位置 |
| 扎根 | 在细分领域建立影响力 |

   - 头部资源碾压

### 品类创新机会

- 开创品类，成为头部
- 案例：喜茶开创奶茶新品类
- 享受头部红利

### 第六章：发现机会 - 发散与收敛

#### 核心概念

1. **决策 = 发散 + 收敛**
   - 发散：容纳更多情景信息
   - 收敛：用逻辑得出结论

2. **工作坊方法**
   - 第一步：穷举（列460件事）
   - 第二步：分线索（4组各找一条）
   - 第三步：找机会点（20个机会paper）
   - 第四步：客观校验（用户访谈）

3. **发散阶段**
   - 穷举法：把能想到的所有事都列出来
   - 不要先入为主
   - 颗粒度要细

4. **收敛阶段**
   - 机会paper三栏：
     1. 问题点：不爽、痛、痒
     2. 问题抽象：本质是什么
     3. 解决方案

5. **客观校验**
   - 让用户直接告诉你：这是幻想还是希望
   - 20个机会 → 8个有效
   - 12个被用户直接否定

#### 与IP定位的关联

| 维度 | IP定位中的应用 |
|------|----------------|
| 发散 | 充分列出用户的所有场景 |
| 收敛 | 找到真正的痛点 |
| 校验 | 让用户验证，不要自己猜 |

### 第十章：跨越周期 - 你的基因，你的机会

#### 核心概念

1. **第二曲线**
   - 持续增长的秘密：第一条曲线消失前开始第二条
   - 不是随便抓一个新机会

2. **奈飞案例：三曲线跨越**
   | 曲线 | 时间 | 业务 |
   |------|------|------|
   | 第一曲线 | 1997 | DVD租赁 |
   | 第二曲线 | 2007 | 在线视频 |
   | 第三曲线 | 2012 | 自制内容 |

3. **核心洞察**
   - 用户需求没变：想方便看电影打发时间
   - 企业基因没变：更多内容→更多会员→更多收入
   - 限制条件在变：介质、技术、内容

4. **企业的基因**
   - 来自创始人的好恶、价值判断、风险偏好
   - 18个月形成，难以改变
   - 决定战场、队友、作战方式

#### 与IP定位的关联

| 维度 | IP定位中的应用 |
|------|----------------|
| 核心不变 | 你持续解决什么问题？ |
| 基因 | 你的内容/风格基因是什么？ |
| 跨越 | 如何找到第二增长点？ |

### 第十二章：设计模式 - 增强回路和调节回路

#### 核心概念

1. **四种连接**
   | 连接 | 说明 |
   |------|------|
   | 因果链 | 因为...所以... |
   | 增强回路 | 越...越...（正反馈） |
   | 调节回路 | 节能、懒（负反馈） |
   | 滞后反馈 | 效果来得慢 |

2. **增强回路**
   - 持续获得正反馈
   - 案例：Costco
     - 会员省钱 → 更多会员 → 供应链更强 → 更大的价格优势

3. **调节回路**
   - 永远存在
   - 节能是生物本能
   - 减肥痛苦：因为对抗本能

4. **滞后反馈**
   - 反馈来得慢
   - 减肥、体重不立刻变化
   - 产品优化、不会立刻爆红

5. **心理能量**
   - 核心资源
   - 心理能量破产 → 事业破产
   - 靠模式而非意志力

#### 与IP定位的关联

| 维度 | IP定位中的应用 |
|------|----------------|
| 增强回路 | 什么让你越做越强？ |
| 调节回路 | 什么在消耗你？ |
| 模式设计 | 如何让用户持续跟随？ |
