# IP定位Pro - 深度定位系统

## 使用场景

当用户希望认真做IP定位、不是随便玩玩，而是希望认真思考并获得可执行的定位报告时使用。

## 核心理论框架

基于梁宁老师三本书籍课程构建的IP定位框架：

```
价值 → 共识 → 模式 → 价值（循环）
```

### 一、真需求：商业价值三角

**产品价值 = 功能价值 + 情绪价值 + 资产价值**

| 价值类型 | 定义 | IP中的应用 |
|---------|------|------------|
| 功能价值 | 解决实际问题的能力 | 你能帮用户解决什么具体问题？ |
| 情绪价值 | 满足情感需求 | 用户关注你时感受到什么情绪？ |
| 资产价值 | 可积累的资产 | 用户能从你这里获得什么可持续资产？ |

### 二、用户画像：大明笨笨小闲

| 用户类型 | 特征 | 内容策略 |
|---------|------|----------|
| 大明 | 需求明确，主动搜索 | 干货、教程、解决方案 |
| 笨笨 | 边逛边看，决策模糊 | 种草、案例、对比 |
| 小闲 | 打发时间 | 娱乐、故事、陪伴 |

### 三、关系层次：需要→喜欢→认同→归属

- **需要**：有用
- **喜欢**：情感连接
- **认同**：价值观一致
- **归属**：社群身份

### 四、增长三级火箭

- 一级火箭：高频流量入口（获客）
- 二级火箭：沉淀用户场景（留存）
- 三级火箭：商业变现（转化）

### 五、点线面体战略

- 点：个人IP
- 线：行业/赛道
- 面：平台/生态
- 体：时代趋势

### 价值定位模块

**V1. 你解决的真实问题是什么？**
- 用户遇到什么具体问题？
- 这个问题不解决会怎样？（恐惧）
- 解决这个问题有什么门槛？

**V2. 你提供什么功能价值？**
- 具体的可交付成果是什么？
- 和同类竞品相比的优势？
- 替代方案是什么？（用户为什么不找别人？）

**V3. 你提供什么情绪价值？**
- 用户关注你时感受到什么情绪？
  - 愉悦？温暖？安全感？归属感？优越感？
- 你让用户逃离什么负面情绪？
- 你给用户带来什么正面情绪？

**V4. 你提供什么资产价值？**
- 用户关注你能获得什么可积累的资产？
  - 粉丝？人脉？技能？认知？圈层？
- 这个资产可持续吗？可变现吗？

## 对话引导流程

当用户启动此skill时，按以下流程引导：

### 第一步：说明用途

"这是一个基于梁宁老师真需求理论的深度IP定位系统。完成问卷后，你会得到一份完整的IP定位报告，包括价值定位、用户画像、共识建立和增长模式四个维度的分析。

请按模块依次填写，越详细定位越精准。如果不确定某项，可以先跳过，我会帮你分析。"

### 第二步：引导填写

按模块依次引导：
1. 先填基础信息（5分钟）
2. 再填价值定位（10分钟）
3. 然后用户定位（10分钟）
4. 接着共识定位（5分钟）
5. 最后模式定位（5分钟）

### 第三步：生成报告

用户完成后，AI按照输出格式生成完整报告。

---

## 注意事项

1. 问卷可以分多次完成，AI会记住进度
2. 某些问题如果用户不确定，可以给出选项让其选择
3. 报告中要给出具体的可执行建议，不只是分析
4. 每个模块都要有结论和行动建议
5. 最后要总结3个核心优势和3个核心挑战
