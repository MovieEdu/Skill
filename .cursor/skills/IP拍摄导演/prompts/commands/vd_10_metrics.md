# /vd_10_metrics - 复盘数据表

在 **项目根/outputs/{时间戳}-{主题概括}/** 下生成或补全 **09_metrics.md**。运行目录 = 用户指定 或 **项目根/outputs/** 下最近包含 **00_plan.md** 的子目录。可读取该目录下 08_revision_notes.md、PUBLISH.md 作为输入。模板见 **skill 内 templates/09_metrics_template.md**（内容复制到输出目录，不写入 skill 目录）。

## 必须生成/包含

- **基础数据**：3 秒留存、完播、互动率、关注率（用户未提供则留空并标注「发布后填」）
- **评论洞察**：最高频误解点 1 条、最高频共鸣点 1 条（用户未提供则留空并标注「发布后填」）
- **下次只改一个**：本条暴露的 1 个问题 + 下次具体改法 1 句（可结合 08_revision_notes 生成建议）

## 文件契约

- 输出：**项目根/outputs/{时间戳}-{主题概括}/09_metrics.md**（若该目录不存在则按 brief 或最近含 00_plan.md 的目录确定运行目录）。
- 禁止在 skill 目录下写入
- 若 09_metrics.md 已存在，可在此基础上补全或更新，而非覆盖用户已填数据
