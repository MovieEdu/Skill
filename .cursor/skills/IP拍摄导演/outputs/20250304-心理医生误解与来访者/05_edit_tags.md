# 05_edit_tags.md

**目标**：从逐字稿按标签提取可剪片段，供剪辑师定位与装配。  
**输入**：`inputs/transcript.md`（拍后逐字稿，含时间戳）。  
**输出**：本文件 + `06_assembly_map.md`。  
**6 发动机自检**：✓ 说服（金句与案例锚）✓ 结构（钩子-冲突-机制-动作）✓ 叙事（尿裤案例、胶片投射类比）✓ 认知（抽象词有翻译句）✓ 交付（追问逼出金句）✓ 影响（金句可做标题/评论引导）
