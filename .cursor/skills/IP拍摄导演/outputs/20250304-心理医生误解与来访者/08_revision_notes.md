   - 发动机：说服（立场明确）+ 结构（钩子/抓手）。

## 最弱环节 1–2 个

1. **证据锚偏单一**（对应 quality_gate：至少 2 类证据锚）  
   - 本条主要靠「亲历/案例」（尿裤、飞机上黑人、程顺恋爱被甩）和「权威/专业」（夕语身份）；没有电影场景、也没有数据。若做硬/燃向，可补：一部影视里「谁才是病人」的片段，或一句靠谱的统计数据（如就诊者中比例）。  
   - 改进：brief 里明确「电影里 3 个可说的场景」或「1 个数据锚」，现场追问时点出一部片/一个数。

2. **动作与边界可更前置、更短**  
   - 「看功能是否受损」「改心里的剧本」在逐字稿后段才集中出现，且和前面信任/投射的讨论略粘连。60s 版里动作落点清晰，但 90s/120s 若再剪一条「纯自检/纯动作」版，可单独录一句「怎么判断该不该看医生？就看功能」+「改自我认同的剧本」做结尾固定句。  
   - 改进：导演在 2/3 处主动追问「那普通人怎么判断自己要不要来？」「你改剧本具体指什么？」逼出一句 10 字内的动作+一句边界。

## 下一次导演提问怎么改（5 条更锋利追问）

1. **钩子后立刻锁机制**：在「真正有病的人不会来」之后马上问：「用一句话说，心理医生到底在做什么？不能解决问题，那在解决什么？」逼出「让你看见自己 / 找自己的资源」一句。

2. **尿裤案例前先给结论**：在讲职场女性之前先问：「有没有一个例子，看起来问题没消失，但人已经好了？」再让夕语讲尿裤，这样剪辑时可以先结论再案例，结构更稳。

3. **投射之后追问动作**：「改心里的剧本」——追问：「如果观众只做一件事，你希望是哪一件？自我认同上具体怎么改？」逼出「我值得 / 会有人喜欢」这类可执行一句。

4. **边界单独问一遍**：「什么时候必须去医院、什么时候找咨询师就够？」「功能受损」的标准能不能再举一个日常例子（如睡不着、不想出门几天算）？

5. **人味收尾提前设计**：「来者不拒去者不留」「你跟我一样」很好；下次可再问一句：「你希望观众走出这条视频时，心里多一句什么话？」收在「你没病」或「反应才是问题」上，便于 60s 结尾统一。
