# /vd_01_topic - 拍前选题手卡

根据 **项目根/outputs/inputs/brief.md** 生成 **项目根/outputs/{时间戳}-{主题概括}/01_topic_card.md**。运行目录 = 用户指定 或 **项目根/outputs/** 下最近包含 **00_plan.md** 的子目录；若不存在则按 brief 解析时间戳+主题概括（时间戳=当日 YYYYMMDD，主题概括=brief「话题一句话」前20字，去掉路径非法字符）创建该目录，并将 **项目根/outputs/inputs/brief.md** 备份到该目录 **00_brief.md**。**禁止在 skill 目录下写入。**

## 必须生成

- **钩子**（3 条短句开头，燃模式优先反差+挑衅式提问）
- **评论区争议句**（3 条）
- **立场句**（承认+反转/补关键变量）
- **观众收益**（情绪/认知/行动各 1 句）
- **结尾评论引导问句**（1 句）

并做 **6 发动机自检**（勾选+一句理由）。

## 文件契约

- 输出文件顶部必须写：目标 / 输入 / 输出 / 6 发动机自检
- 只写入 **项目根/outputs/{时间戳}-{主题概括}/01_topic_card.md**（输出目录规则见上），不覆盖其他输出

## 参考
