# /vd_03_questions - 导演问题脚本（5 层梯子 + 追问 + 救场）

读取 **项目根/outputs/{时间戳}-{主题概括}/01_topic_card.md** + **项目根/outputs/{时间戳}-{主题概括}/02_anchor_pack.md**。运行目录 = 用户指定 或 **项目根/outputs/** 下最近包含 **00_plan.md** 的子目录。
生成 **项目根/outputs/{时间戳}-{主题概括}/03_director_questions.md**。

## 必须包含

### A) 【1 页导演手卡】（现场只看这一页）

- 5 层梯子，每层 2-3 个主问题（口语化）
- 每层 1 句「转场句」
- **每层至少 1 次「逼金句」动作**（见 D 模块）
- 金句逼出器 3 条
- 救场句 6 条（拉回/逼具体/逼边界）

### B) 【完整版问题脚本】

- 每个主问题配 2 个追问：
  - 追问1：逼画面（谁、在哪、做了什么）
  - 追问2：逼机制（关键变量一句话）
- **燃模式**要求：
  1) 每层至少 1 个短句连击问题（例如：「你怕什么？你赌什么？你保什么？」）
  2) 机制句 ≤20 字
  3) 必须逼出「一步动作+一条边界」

### C) 【批判性追问模块】（每条视频至少用到 3 类，口语化、像人一样追问）

- **定义追问**：你说的「____」具体指什么行为？反例是什么？
- **假设追问**：你这个判断默认了什么前提？如果前提不成立呢？
- **证据追问**：你最强的证据/场景是哪一个？
- **替代原因**：除了你说的原因，还有没有别的解释？
- **遗漏信息**：这件事我们最容易忽略的变量是什么？
- **可证伪**：什么情况出现，你会愿意修正观点？

（来源：《学会提问》式训练——透彻 = 定义清楚 + 假设清楚 + 证据清楚 + 替代解释清楚；不要写成学术审问，必须是导演像人一样追问。）

### D) 【金句逼出器】（每层至少触发 1 次）

- **只留一句**：如果只能留一句给观众，你留哪句？
- **再短**：再短一半，不许加解释。
- **更像人话**：去掉抽象词，换成一个具体动作/画面。

（燃 = 节奏 + 句子密度；没有强制「收句」，现场会变成一段段叙述，剪不出能当封面字幕的句子。）

并做 **6 发动机自检**。
