# Director Quality Gate v1.0

上线前必须全部勾选通过。

- [ ] 至少2类证据锚：电影场景 +（亲历/权威/数据）任选其一
- [ ] 现场必须拿到：HOOK + 冲突/代价 + 关键变量一句话 + 抓手(3个) + 动作(1个) + 边界(1条) + 人味瞬间
- [ ] 抽象词（人味/纯粹/灵活）都有「翻译句+例子」
- [ ] 机制能被一句话转述（不超过20字更好）
- [ ] 动作是可执行的（不是口号）
- [ ] 结尾能把人扶起来（燃但不鸡汤）
- [ ] **事实核查（默认）**：所有「数字/研究/剧情细节/因果断言」均已核查或降级表述（未核实则标「需核查」或改为「很多人认为…」等）
